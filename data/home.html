This is a simple javascript function to convert a json array to an html table

json array should be in a below format

[{'key1':'value1_A', 'key2':'value2_A'},{'key1':'value1_B', 'key2':'value2_B' },{'key1':'value1_C', 'key2':'value2_C' }]

This is an array of javascript object and each object is a list of key & value pairs

Some feature of the code

  • You can have many different types of keys in each object and can have diferent types of values as well.
  • A value can be an objects as well. In this case the created table will have a child table in place of value.

How to use

After including the javascript file in you code just call the below function

makeTable(obj_for_table);

obj_for_table is a variable containing the array of abjects.

It will return a string having the html of the table, you can display it as below in your page

$('#table_container').html(makeTable(json_arr));
